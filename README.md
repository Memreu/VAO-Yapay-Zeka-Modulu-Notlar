## 🚀 Nasıl Kullanılır?

Bu repoda derslere ait teorik notlar **PDF** formatında, uygulamalı kod örnekleri ise **Jupyter Notebook (.ipynb)** formatında yer almaktadır.

### 📄 Ders Notları (PDF & HTML)
* **Görüntüleme:** Ana dizinde yer alan PDF (örneğin `1. ders.pdf`) veya HTML formatındaki dosyalara tıklayarak teorik notları doğrudan GitHub üzerinden tarayıcınızda inceleyebilirsiniz.
* **İndirme:** Dosyaların sağ üst köşesinde bulunan **"Download raw file" (İndir)** butonunu kullanarak içerikleri cihazınıza indirebilir ve çevrimdışı olarak çalışabilirsiniz.
* **Format Seçimi:** Notlara hem PDF hem de tarayıcı dostu HTML formatında erişerek size en uygun okuma deneyimini seçebilirsiniz.

### 💻 Pratik Uygulamalar (.ipynb)
Derste işlenen kod uygulamaları `pratik-uygulamalar` klasörü içerisinde bulunmaktadır. Bu dosyaları incelemek ve çalıştırmak için aşağıdaki yöntemlerden birini seçebilirsiniz:

#### Yöntem 1: Google Colab ile Çalıştırma (Kurulum Gerektirmez - Önerilen)
Bilgisayarınıza herhangi bir program kurmadan, tarayıcınız üzerinden kodları anında çalıştırabileceğiniz en pratik yöntemdir:
1. [Google Colab](https://colab.research.google.com/) adresine gidin.
2. Açılan pencerede **"GitHub"** sekmesini seçin.
3. Arama çubuğuna bu reponun linkini yapıştırın veya GitHub kullanıcı adınızı aratın.
4. Çıkan listeden çalıştırmak istediğiniz `.ipynb` dosyasını seçerek hemen kodlamaya başlayabilirsiniz.

#### Yöntem 2: Kendi Bilgisayarınızda Çalıştırma (Lokal)
Dosyaları kendi bilgisayarınızda çalıştırmak isterseniz:
1. Bu repoyu bilgisayarınıza klonlayın (`git clone`) veya **"Code"** butonuna basarak ZIP olarak indirin.
2. Bilgisayarınızda Python ve Jupyter Notebook'un (Anaconda da tercih edilebilir) kurulu olduğundan emin olun. (Alternatif olarak **Visual Studio Code** kullanabilir ve Jupyter eklentisini kurabilirsiniz).
3. İndirdiğiniz `.ipynb` dosyasını bu programlardan biriyle açın ve kod hücrelerini sırasıyla çalıştırın (Genellikle `Shift + Enter` kısayolu ile).
